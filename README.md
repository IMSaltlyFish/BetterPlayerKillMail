# BetterPlayerKillMail

一个为 **Folia / Paper** 核心设计的高性能 **死亡邮件** 插件。

当玩家死亡时，自动记录死亡信息（时间、死因、掉落物等），并允许其他玩家通过 GUI 查看详细的死亡报告。

灵感来源于 EVE Online。

> 项目仍在施工中 🚧

## ✨ 特性

- **完整的死亡记录**  
  记录受害者、凶手（如果有）、击杀方式、死亡时间、死亡消息、掉落物品快照。

## 📋 计划中的内容

- 更加具体的战斗快照
- 完整的权限控制
- i18n 支持
- SQLite、MySQL 数据库支持

## 📥 安装

1. 下载 `BetterPlayerKillMail.jar`
2. 将文件放入服务器的 `plugins/` 文件夹
3. 重启服务器（或使用 `plugman` 热加载）
